1. download helm binary from: https://github.com/helm/helm/releases
2. copy helm[.exe] to /usr/local/bin or c:/Windows/System32
3. helm repo add stable https://charts.helm.sh/stable
4. helm repo update
5. helm install my-redis bitnami/redis --set password="secretpass"
6. kubectl get all
7. helm upgrade redis-prod stable/redis --set cluster.slaveCount=5 --set password="secretpass" --generate-name
   helm upgrade redis-prod stable/redis -f values.yaml
8. helm delete redis-prod

=== Installing MySQL using helm =======
1. helm repo add bitnami https://charts.bitnami.com/bitnami
2. helm install my-release bitnami/mysql --set password="secretpass"
